⚙️ GitHub Token 設定
需要 repo 權限，有效期限建議 30 天。Token 保存在本地，下次自動填入。
📖 如何申請 GitHub Token？
  1. 登入 GitHub
    造訪 github.com 並登入帳號。
  2. 進入 Developer settings
    頭像 → Settings → 左側最下方 Developer settings
    或直接造訪 github.com/settings/tokens
  3. 建立新 Token
    Personal access tokens → Tokens (classic)
    點擊 Generate new token → Generate new token (classic)
  4. 填寫設定
    Note：例如 GitHub Pages 部署工具
    Expiration：選擇 30 days
    Scopes：只勾選 ☑ repo，其他不要勾選
  5. 複製 Token
    點擊 Generate token
    ⚠️ Token 只顯示一次！格式為 ghp_xxxxxxxxxxxx，離開後無法再查看。
  6. 貼上 Token
    將 Token 貼入上方欄位，點擊「儲存」即可。下次開啟頁面會自動帶入。
